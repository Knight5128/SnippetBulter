关于文本片段管理器

文本片段管理器是一个高效的文本片段存储和管理工具，帮助您快速存储、组织和检索常用的文本片段。

版本: 1.0

使用技术: AutoHotkey v2, HTML, CSS, JavaScript

使用指南

快速使用:

  • 按下全局快捷键 (默认Ctrl+Shift+Space) 唤出片段选择器
  • 选择或搜索您需要的片段
  • 双击选择片段，内容会自动复制到剪贴板
  • 按ESC关闭选择器

片段管理:

  • 在管理面板中添加、编辑和删除片段
  • 创建分组以更好地组织您的片段
  • 根据需要修改应用设置
