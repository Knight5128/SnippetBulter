# Text Snippet Manager

一个高效管理和快速使用文本片段的桌面工具，旨在提升您的文本输入效率。

## 简介 (Overview)

您是否经常需要在聊天、邮件、编程或写作中重复输入某些固定的文本段落、代码片段、邮箱地址或常用语？Text Snippet Manager 就是为此而生！

它允许您：

*   **存储和组织** 您常用的文本片段。
*   通过**分组和子分组**清晰地管理它们。
*   设置一个**全局快捷键**，随时随地快速唤出一个简洁的界面。
*   在唤出的界面中**浏览、搜索(未来计划)**并**双击**需要的片段，它将自动复制到您的剪贴板。
*   通过直观的管理面板**轻松添加、编辑和删除**您的文本片段。
*   (可选) 集成并启用额外的实用工具脚本。

本应用基于 AutoHotkey 构建，力求轻量、快速且实用。

## 主要特性 (Features)

*   **快捷唤出:** 通过自定义全局快捷键 (默认待定，可在设置中修改) 快速打开片段选择界面。
*   **卡片式布局:** 在选择界面中，以清晰的卡片形式展示文本片段。
*   **分组管理:** 支持创建分组和子分组，方便组织大量片段，并可在选择界面中展开/折叠。
*   **快速复制:** 双击选择界面中的任意片段卡片，即可将其内容复制到系统剪贴板。
*   **后台运行:** 应用在后台静默运行，只在需要时出现，不打扰您的工作流程。提供系统托盘图标进行访问和退出。
*   **可视化管理面板:**
    *   添加、编辑、删除文本片段。
    *   创建、重命名、删除分组。
    *   拖拽调整片段和分组的顺序与层级。
*   **集成可选脚本:** (可在管理面板中开启/关闭)
    *   **Easy Window Dragging:** (来自 `easy-window-dragging.ahk`) 可能提供更便捷的窗口拖动方式（例如按住某个键在窗口任意位置拖动）。
    *   **On-Screen Keyboard:** (来自 `on-screen-keyboard.ahk`) 可能提供一个简单的屏幕键盘功能。
*   **数据本地存储:** 所有片段和配置都安全地存储在本地文件中 (JSON 格式)。

## 安装与运行 (Installation & Usage)

**对于开发者或测试者 (从源码运行):**

1.  **安装 AutoHotkey:** 确保您已安装 [AutoHotkey v2](https://www.autohotkey.com/)。
2.  **获取源码:** 下载或克隆本仓库代码。
3.  **运行:** 双击 `Text-Snippet-Manager.ahk` 文件。应用将在后台运行，并在系统托盘区域显示一个图标。

**对于最终用户 (使用打包后的 `.exe`):**

1.  **下载:** 从发布页面下载最新的 `.exe` 安装包或独立可执行文件。
2.  **运行/安装:**
    *   如果是独立 `.exe` 文件，直接双击运行即可。
    *   如果是安装包，请按提示完成安装。
3.  应用将在后台运行，并在系统托盘区域显示一个图标。

## 使用说明 (How to Use)

1.  **运行应用:** 启动 `Text-Snippet-Manager.ahk` 或其打包后的 `.exe` 文件。
2.  **添加/管理片段:**
    *   右键单击系统托盘图标，选择 "打开管理面板"。
    *   在管理面板中，您可以：
        *   点击 "添加片段" 按钮创建新的文本片段。
        *   点击 "添加分组" 按钮创建新的分组。
        *   选中现有片段或分组进行编辑或删除。
        *   拖拽片段或分组来调整顺序或归属关系（例如，将片段拖入分组）。
        *   在设置区域配置快捷键，或启用/禁用集成的额外脚本。
3.  **唤出选择界面:** 按下您在管理面板设置的全局快捷键 (例如 `Ctrl+Shift+Space`，具体待定)。
4.  **选择并复制片段:**
    *   在弹出的选择界面中，浏览您的片段。
    *   点击分组标题可以展开或折叠该分组下的片段。
    *   找到您需要的片段后，**双击**该片段的卡片。
    *   该片段的内容现在已经复制到您的剪贴板中。
5.  **粘贴:** 在您需要的位置 (聊天窗口、编辑器等) 按 `Ctrl+V` 粘贴即可。
6.  **退出应用:** 右键单击系统托盘图标，选择 "退出"。

## (可选) 集成脚本说明

*   **Easy Window Dragging:** 启用后，可能允许您通过按住特定修饰键（如 `Alt` 或 `Ctrl`）并在窗口的任意位置单击并拖动鼠标，来移动该窗口。具体行为取决于 `easy-window-dragging.ahk` 的实现。
*   **On-Screen Keyboard:** 启用后，可能会通过某个快捷键或托盘菜单选项调出一个简单的屏幕键盘界面。具体行为取决于 `on-screen-keyboard.ahk` 的实现。
